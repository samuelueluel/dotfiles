# System Setup - Fedora Atomic (Niri)

This guide covers specialized manual configurations for the Fedora Atomic (BlueBuild) host.

## 1. Swap Configuration (16GB Swap File)

By default, Fedora uses an 8GB zRAM device. For workloads requiring more physical swap space, we replace it with a 16GB file on `/var`.

### Verification (Before Applying)
Check the current swap status on the host:
```bash
swapon --show
```
If you see `/dev/zram0`, zRAM is active. If you see `/var/swapfile`, the swap file is already set up.

### Application Steps
If you need to (re)create the 16GB swap file:

1. **Disable zRAM Generator:**
   ```bash
   sudo touch /etc/systemd/zram-generator.conf
   sudo swapoff /dev/zram0
   ```

2. **Create the 16GB Swap File:**
   ```bash
   sudo fallocate -l 16G /var/swapfile
   sudo chmod 600 /var/swapfile
   sudo mkswap /var/swapfile
   sudo swapon /var/swapfile
   ```

3. **Ensure Persistence:**
   Add the following line to `/etc/fstab` if it's not already there:
   ```text
   /var/swapfile none swap defaults 0 0
   ```

### Verification (After Applying)
Ensure the file is active and the correct size:
```bash
swapon --show
```
